# CLARA Security - Android Companion App

Modern Material 3 tasarımlı güvenlik kontrol uygulaması.

## Özellikler
- Dashboard: Tehdit özeti, tarama durumu
- Bildirimler: Gerçek zamanlı tehdit uyarıları
- Ayarlar: Modül yapılandırması
- Tehdit Geçmişi: Son tespit edilen tehditlerin listesi
- App Lock: Uygulama kilitleme yönetimi

## Teknoloji
- Kotlin
- Jetpack Compose
- Material 3
- Unix Domain Socket IPC

## Gereksinimler
- Android 10+ (API 29+)
- KernelSU + CLARA Module kurulu
